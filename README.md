# AuditMonitor

**AuditMonitor** is a PCF (Power Apps Component Framework) component designed to simplify the review and visualization of record audit history in Microsoft Dynamics 365 / Dataverse.

The component presents field-level changes in a clear and structured way, including information such as date, time, responsible user, executed action, and changed values. Its purpose is to provide a more practical and transparent experience for monitoring audit records directly from Dynamics 365 forms.

AuditMonitor provides a modern interface, search support for changed fields, and availability in Portuguese, English, French, Spanish, German, and Danish.

To use the component, associate it with a text field on the form of the desired entity.
